- ## Cómo publicar en GitHub Pages
1. Crear repo nuevo
2. Subir `index.html` y `data.json`
3. GitHub → Settings → Pages
4. Source: Deploy from a branch
5. Branch: `main` y carpeta `/ (root)`
6. Guardar
